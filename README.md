# Med-Cure

MedCure is a full-stack healthcare web application that allows users to order medicines online and enables doctors to manage digital prescriptions securely.

# Features

- User Signup & Login Authentication
- Session Management using Express Session
- Online Medicine Ordering System
- Search & Category Filtering for Medicines
- Shopping Cart Functionality
- Prescription Verification for Restricted Medicines
- Doctor Dashboard for Adding Prescriptions
- Prescription Expiry Validation
- MongoDB Database Integration
- Responsive User Interface

# Installation & Setup

1️.Clone the Repository
git clone https://github.com/your-username/MedCure.git

2️. Navigate to Project Folder
cd MedCure

3️. Install Dependencies
npm install

4️. Start the Server
node server.js

5️. Open in Browser
http://localhost:5000

# User Roles

# User
- Signup/Login
- Browse medicines
- Add medicines to cart
- Upload and use prescriptions

# Doctor
- Login to doctor dashboard
- Add prescriptions for users
- Set prescription expiry dates
